[FINATEC AG (now part of CPB Software)](https://www.cpb-software.com/) was an enterprise software company, specialised in creating software for securities trading used in TOP10 of the banks in Germany.

It started with an security trading system called SECTRAS, written in Cobol and already really popular in the banks. We modernised it using java back-end and ORACLE database to bring the functionality provided of SECTRAS as [iSectras product](iSectras.md). It was also hugely popular and embraced by TOP10 of the banks in Germany.
